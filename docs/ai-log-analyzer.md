# 🤖 AI Log Analyzer (AIOps Foundation)

## Overview

The platform includes an AI-assisted log analysis component designed to detect operational anomalies automatically.

Application logs generated by containers are analyzed to identify potential incidents.

---

## Logging Flow

```text
Application → Container Logs → Host Volume → AI Analyzer
```

---

## Analyzer Capability

* Reads recent application logs
* Detects ERROR and WARNING patterns
* Generates operational summary
* Highlights potential incidents

---

## Execution

```bash
python aiops/log_analyzer.py
```

---

## Example Output

```text
AI LOG ANALYSIS REPORT
Errors detected: 1
Potential Incident Detected
```
